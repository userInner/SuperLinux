# AI Learning Hub - 人工智能学习中心

一个现代化的AI学习网站，提供人工智能相关的课程、技术挑战和学习资源。

## 项目特点

### 🎯 核心功能
- **响应式设计**: 完美适配桌面、平板和手机设备
- **交互体验**: 平滑滚动、动画效果、悬停交互
- **学习内容**: 涵盖机器学习、深度学习、NLP、CV等AI核心领域
- **技术挑战**: 展示当前AI面临的技术问题与解决方案
- **学习路径**: 个性化学习路线规划

### 🛠️ 技术栈
- **前端**: HTML5, CSS3, JavaScript (ES6+)
- **样式**: CSS变量、Flexbox、Grid布局
- **动画**: CSS动画、JavaScript动画
- **图标**: Font Awesome 6
- **字体**: Google Fonts (Poppins, Inter)

### 📁 项目结构
```
ai-learning-website/
├── index.html          # 主页面
├── css/
│   └── style.css      # 样式文件
├── js/

```

## 🔧 开发指南

### 添加新课程
1. 在 `index.html` 的 `.course-grid` 中添加新的 `.course-card`
2. 更新课程图标、标题、描述和元数据
3. 样式会自动适配

### 添加新挑战
1. 在 `index.html` 的 `.challenge-grid` 中添加新的 `.challenge-card`
2. 设置挑战图标、标题、描述和标签
3. 标签会自动应用样式

### 自定义样式
1. 修改 `css/style.css` 中的CSS变量
2. 调整颜色、间距、字体等
3. 添加新的动画效果
